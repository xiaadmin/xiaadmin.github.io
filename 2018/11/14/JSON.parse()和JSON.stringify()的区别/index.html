nodejs 2018/11/14

JSON.parse()和JSON.stringify()的区别

JSON.parse()从一个字符串中解析出JSON对象

例如
{“students”:[{“name”:”小明”,”sex”:”女”,”age”:23,”hobby”:”打台球”}]}

执行之后的结果
name:”小明”,sex:”女”,age:23,hobby:”打台球”

JSON.stringify()从一个对象中解析出字符串

同上面相反

例如
var students={name:”小明”,sex:”女”,age:23,hobby:”打台球”}

执行之后的结果
{“students”:[{“name”:”小明”,”sex”:”女”,”age”:23,”hobby”:”打台球”}]}
