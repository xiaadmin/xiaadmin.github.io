—# Hexo+github搭建个人博客

@(博客搭建)[作者:夏鑫|编辑Markdown]

###什么是Hexo?
HexoHexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

###什么是Markdown?
Markdown是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面。

搭建步骤

###1.软件的的安装
—安装nodejs
—安装Git

####1>下载并安装nodejs
——下载nodejs安装包
——安装nodejs
——安装完成之后WIN+R 输cmd打开控制台
——输入node -v查看node版本 如果有则node安装成功

——输入npm -v查看npm 版本 如果有则npm安装成功

####2>下载并安装Git
——官网下载Git
——安装Git
——验证是否安装成功（WIN+R 输入cmd进入控制台输入git –version）

安装完成之后，右键鼠标就可以看见Git GUI Here 和 Git Bash Here

####3>安装Hexo
——在任意位置打开Git Bash Here
——输入npm install -g hexo-cli

由于连接国外服务器 ，所以安装过程比较慢，当然我们也可以使用淘宝镜像进行安装，使用如下命令安装淘宝镜像

安装完之后使用如下命令安装速度就会变快

–安装完之后时候hexo -v 验证是否安装成功

###2.Hexo生成本地博客

####1>创建本地文件夹
这个文件夹将放Hexo生成的源文件，文件名称可以随意。

####2>运行Git Bash
在创建好的文件夹下右键运行Git Bash

####3>初始化Hexo
–输入 hexo init

完成后打开文件夹，此时文件夹已生成了一些文件

–输入npm install 安装所需组件

####4>修改配置文件信息
–打开_config.yml文件
修改其中的title ，author 等信息

####4>生成博客
–Git Bash控制台输入hexo g

####5>本地部署测试
–Git Bash控制台输入hexo s

–浏览器中输入红的方框中的地址就可以看到自己的博客了
本地博客的生成已经到此结束
